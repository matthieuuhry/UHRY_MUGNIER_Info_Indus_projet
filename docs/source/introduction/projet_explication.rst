Presentation du projet
======================

L’objectif de ce projet et de faire fonctionner la plateforme robotique, pentographe avec ROS2 sur un Raspberry Pi (Pi5) et documenter le projet.

.. figure:: img/real_system_photo.png
   :align: center
   :width: 60%
   
Le projet consiste donc à:

1. Décrire la plateforme mécanique
2. Décrire le matériel électronique, carte dynamixel avec les liens sur les documentations techniques (datasheets)
3. Créer la rerprésentation mécanique du pantographe dans un fichier URDF
4. Visualiser le résultat avec RVIZ
5. Décrire la dynamique et donner les équations pour piloter la position de l’organe terminal du pentographe: comment
6. Créer un package ROS2 pour contrôler le pantographe
7. Créer des tests et documenter les tests
8. Créer un code pour dessiner avec le pentographe
9. Tester en réel avec le pantographe
10. Créer un code de simulation pour le pentographe
11. Tester en simulation avec Gazebo et RViz

La documentation sera publiée sur notre github.
